# CineScribe

### App to provide movie reviewers and vloggers a tool to record fresh ideas, thoughts, and feeling about a movie that was just seen.

### Developers:
  * Jeff Santana
  * Marlon Raskin
  
### Technologies:
  * UIKit
  * CoreData
  * Firebase
  * Pull to Reach
  * The Movie database API
  
### Target:
  iOS 13
